JSON-LD Schema markup + full SEO analysis engine with per-page scoring and site-wide reports.

== Description ==
Grapevine SEO gives you two powerful tools in one plugin:

1. **Schema Markup** — Outputs Google Rich Results-compliant JSON-LD schema for Article, FAQ, HowTo, Product, Event, Recipe, LocalBusiness, and more.
2. **SEO Analysis Engine** — Scores every page across 22 SEO checks in 5 categories (Content, Title & Meta, Schema, Social, Keyword). Shows passes, warnings, failures, and fix suggestions.

== Features ==
* Per-page schema override with 11 schema types
* Full SEO analysis dashboard with score gauges and distribution charts
* Site-wide "Analyze All" with progress indicator
* Per-page dropdown with live analysis and check cards
* All-pages table with filter/search by score grade
* Meta description character counter
* Open Graph / social tag management
* Google Rich Results compliant (article headline ≤110 chars, Event eventStatus/eventAttendanceMode, Product ratingCount, Recipe image required)
